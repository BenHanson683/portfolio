Database Management Systems


Below are several projects I have completed concerning the construction and design of Database Management Systems. The DMS software that was used to complete these projects was MySQL and SQL Server Management Studio (SSMS).

Ch. 3 Project

This project consists of developing a database and an Entity Relationship Diagram (ERD) for a classic car museum database system. Data was stored in the database for cars, car owners, and the museum. The project consists of the database design including tables, fields, and relationships. All the entity types that can fully describe this database system and its environment are specified, as well as the relation types among them.

Ch. 5 Project:

This project deals with the creation of a movie theater database using SSMS. Through the Table Designer, tables and keys were created. Also, using the editor, test data was added to the tables. An ER diagram was created with the addition of relationships. Potential uses, actors, and users were all defined in this project with additional table designs.

Ch. 6/7 Project:

In this project, a hotel business database was created consisting of four main tables. This database was created based upon several predetermined requirements including primary and foreign keys. SQL INSERT statements were also utilized to insert rows into the tables. An INSERT trigger was deployed as well to perform calculations.
